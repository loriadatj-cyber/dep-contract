# Roadmap

The roadmap is driven by maintainer feedback and real lockfile fixtures.

## 0.1 - npm foundation

- package-lock v2/v3 graph contracts
- source, integrity, and install-script policies
- text, JSON, and GitHub job summary output
- auditable approval reasons

## 0.2 - review ergonomics

- scoped and expiring policy exceptions
- compact reports for large lockfile updates
- optional direct-vs-transitive dependency grouping
- SARIF output

## 0.3 - more ecosystems

- pnpm lockfiles
- Yarn lockfiles
- shared normalized lockfile model

## 1.0 - stable automation

- stable contract schema and migration command
- first-class pull request annotations
- signed release and npm provenance
- documented integration API
